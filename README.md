# fight.py

Простая игра, где тебе надо победить врага
В твоих силах доступны такие действия, как:
1. Атака, 2. Защита и 3. Лечение

Но у врага тоже есть эти возможности!
Попытайся победить его, но будь осторожен - все его действия непредсказуемы!

После битвы можно либо выбрать 1. и повторить бой, либо выйти на 2.

## Инструкция:
1. ```git clone https://github.com/astyarik/fight.py.git```
2. ```python fight.py```
3. Удачной игры!
